Dashboard Metro CDMX

Análisis interactivo de la afluencia de usuarios por fecha, línea y estación.

Descripción del proyecto: este dashboard muestra una prueba de visualización interactiva realizada con Python y Plotly. La gráfica permite explorar el comportamiento de la afluencia del Metro mediante herramientas como zoom, desplazamiento, selección de puntos y visualización de valores al pasar el cursor.

Fuente de datos

Registros históricos de afluencia del Metro CDMX.

Herramientas

Python, Pandas, Plotly, HTML y GitHub Pages.

Objetivo

Publicar gráficas interactivas en una página web desde un repositorio de GitHub.
